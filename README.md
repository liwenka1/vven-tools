## 📦 快速开始

### 环境要求

- **Node.js** 18.x 或更高版本
- **pnpm** 9.x 或 **npm** 10.x
- **现代浏览器** (Chrome 90+, Firefox 88+, Safari 14+)

### 本地部署

```bash
# 1. 克隆项目
git clone https://github.com/liwenka1/vven-tools.git
cd vven-tools

# 2. 安装依赖 (推荐使用 pnpm)
pnpm install
# 或者使用 npm
npm install

# 3. 启动开发服务器
pnpm dev
# 或者使用 npm
npm run dev

# 4. 打开浏览器访问
open http://localhost:3000
```

### 生产部署

```bash
# 构建生产版本
pnpm build

# 启动生产服务器
pnpm start

# 或者使用静态部署
pnpm build && pnpm export
```
